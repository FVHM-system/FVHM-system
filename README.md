# water-management 水资源管理系统


#### 技术栈
vue3 + vuex + Element-Plus + Echarts + vue-router + vite


#### 开发时服务

1.  yarn
2.  yarn dev


#### 注意

使用的 vite 脚手架对开发时的热更新不是很好，如果有页面更新有异常，第一时间尝试 刷新页面 / 重启服务 解决
